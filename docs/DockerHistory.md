# Docker发布历史
包含cuda11和cuda12两个版本镜像
## 开发镜像
开发镜像已配置cuda，bazel等基础环境，但不包含已安装的wheel包，支持用户从源码开始构建/测试

###  0.0.1
```

cuda12: registry.cn-hangzhou.aliyuncs.com/havenask/rtp_llm:cuda12
```
